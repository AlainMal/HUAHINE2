Installation

Comment installer l'application

  1. Il faut, pour l'instant, installer les CANBUS.
    • Choisissez un fichier éxistant.
    • Cliquer sur 'Enregistrer'.

  2. Cliquez sur importer le fichier texte dans la barre d'outils.
    • Vous avez la possibilité de valider, ou de choisir les numéros des lignes.

  3. Vous avez à présent la possibilité de cliquez sur la table.
    • Vous verrez alors les résultats de la conversion en NMEA 2000.

Information : Vous verrez sur le fichier .csv les différentes valeurs des PGN à chaques lignes.

Formats supportés

  • Fichiers .csv
